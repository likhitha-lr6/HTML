17. A semantic element clearly describes its meaning to both the browser and the developer. Examples of **non-semantic** elements: **`<div>` and `<span>`** - Tells nothing about its content. Examples of **semantic** elements: **`<img>`, `<table>`, and `<article>`** - Clearly defines its content.
18. A URL is another word for a web address. A URL can be composed of words (e.g. google.com), or an Internet Protocol (IP) address (e.g. 192.68.20.50).
19. An HTML form is used to collect user input. The user input is most often sent to a server for processing.
